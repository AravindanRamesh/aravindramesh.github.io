Customer Churn Prediction

This project involved developing a machine learning model to predict customer churn with 80% accuracy.

  • Used Apache Spark for distributed data processing and real-time analytics.
  • Applied Random Forest Classifier to predict customer churn based on historical data.
  • Integrated the solution with a real-time data pipeline to provide timely insights.
  • Achieved an 80% accuracy rate, allowing proactive customer retention strategies.
